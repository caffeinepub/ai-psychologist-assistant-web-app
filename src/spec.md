# AI Psychologist Assistant Web App

## Overview
A multilingual emotional support web application that provides gentle, empathetic conversation through an AI assistant. The app supports Tamil, English, Tanglish (Tamil-English mix), Telugu, Kannada, Hindi, and Marathi, automatically detecting and responding in the user's preferred language and tone.

## Core Features

### Conversation System
- Real-time chat interface with AI assistant
- Automatic language detection and response matching
- Structured conversation flow:
  1. Initial invitation to share concerns
  2. Empathetic emotion reflection
  3. 1-2 calming follow-up questions
  4. Reassurance and emotional grounding tips
  5. Hopeful, soothing conclusion

### Therapeutic Elements
- CBT-style reflective questioning
- Automatic detection of anxiety-related keywords to trigger breathing exercise suggestions
- Gentle encouragement for journaling and self-care practices
- Grounding technique recommendations

### Safety and Ethics
- Built-in disclaimers stating this is emotional support, not medical advice
- No diagnosis or medication prescription capabilities
- Compassionate responses to distress with gentle guidance toward professional help
- Crisis detection for self-harm expressions with appropriate supportive responses

### User Interface
- Soft, calming pastel color scheme
- Minimalistic chat layout
- Smooth text animations for messages
- Human-like typing indicators with realistic pacing
- Optional "Calm Mode" featuring a visual breathing circle for guided breathing exercises

### Language Support
- Multilingual support for Tamil, English, Telugu, Kannada, Hindi, Marathi, and Tanglish
- Tone and language matching based on user input
- Cultural sensitivity in responses

## Technical Requirements

### Frontend
- Chat interface with message history
- Language detection and response formatting
- Breathing exercise visualization (expandable/contractible circle)
- Smooth animations and transitions
- Responsive design for mobile and desktop

### Backend
- AI conversation processing with multilingual capabilities
- Conversation history storage per user session
- Language detection and response generation
- Keyword analysis for therapeutic triggers
- Session management for continuous conversations

## App Identity
The assistant presents itself as a supportive emotional companion, clearly distinguishing itself from professional therapy or medical services while maintaining a warm, safe, and understanding presence.
